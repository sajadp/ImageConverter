دانلود و نصب برنامه
================================================================================

روش 1: استفاده از فایل DMG (توصیه می‌شود)
--------------------------------------------
1. فایل "ImageConverter-DMG.zip" را از GitHub Releases دانلود کنید
2. فایل zip را باز کنید تا "ImageConverter.dmg" بدست بیاید
3. روی "ImageConverter.dmg" دابل کلیک کنید
4. "ImageConverter.app" را به پوشه Applications بکشید
5. با کلیک راست روی DMG، گزینه "Eject" را انتخاب کنید
6. برنامه را از Launchpad یا پوشه Applications باز کنید


روش 2: استفاده مستقیم از فایل .app
--------------------------------------
1. فایل "ImageConverter-macOS.zip" را از GitHub Releases دانلود کنید
2. فایل zip را باز کنید تا "ImageConverter.app" بدست بیاید
3. "ImageConverter.app" را به پوشه Applications منتقل کنید
4. برنامه را از پوشه Applications باز کنید


راه‌اندازی اولیه (مهم!)
================================================================================

وقتی برای اولین بار ImageConverter را باز می‌کنید، macOS این هشدار را نشان می‌دهد:

    "ImageConverter cannot be opened because the developer 
     cannot be verified."

این مشکل طبیعی است برای برنامه‌هایی که از طریق Mac App Store توزیع نشده‌اند.

راه‌حل - گزینه 1 (سریع):
--------------------------
1. روی ImageConverter.app کلیک راست کنید (یا Control+کلیک)
2. گزینه "Open" را از منو انتخاب کنید
3. روی "Open" در پنجره تأیید دوباره کلیک کنید
4. برنامه حالا به طور عادی باز می‌شود


راه‌حل - گزینه 2 (از طریق تنظیمات):
-------------------------------------
1. سعی کنید برنامه را به طور معمولی باز کنید (مسدود می‌شود)
2. بروید به: System Settings → Privacy & Security
3. پایین اسکرول کنید تا پیدا کنید: "ImageConverter was blocked from use"
4. روی "Open Anyway" کلیک کنید
5. در صورت درخواست، رمز عبور خود را وارد کنید
6. روی "Open" در پنجره تأیید کلیک کنید


نحوه استفاده
================================================================================

1. روی "Select Images" کلیک کنید تا تصاویر خود را انتخاب کنید
   - فرمت‌های پشتیبانی‌شده: HEIC، HEIF، PNG، JPG، BMP، TIFF، WEBP

2. فرمت خروجی را انتخاب کنید:
   - WEBP: فرمت مدرن با بهترین فشرده‌سازی
   - JPG: سازگاری جهانی

3. کیفیت را تنظیم کنید (1-100):
   - بالاتر = کیفیت بهتر، حجم فایل بیشتر
   - پایین‌تر = حجم فایل کمتر، کیفیت پایین‌تر

4. روی "Start Conversion" کلیک کنید

5. محل ذخیره تصاویر تبدیل‌شده را انتخاب کنید

6. تمام! تصاویر تبدیل‌شده را در پوشه خروجی بررسی کنید


امکانات برنامه
================================================================================

✓ تبدیل تصاویر HEIC/HEIF به WEBP یا JPG
✓ پردازش دسته‌ای (تبدیل چند تصویر به صورت همزمان)
✓ تنظیمات کیفیت قابل تنظیم
✓ رابط کاربری مدرن و کاربرپسند
✓ تبدیل سریع و کارآمد


نیازمندی‌های سیستم
================================================================================

• macOS 10.13 (High Sierra) یا بالاتر
• 100 مگابایت فضای خالی روی دیسک
• Python runtime (در برنامه تعبیه شده)


رفع مشکلات رایج
================================================================================

مشکل: برنامه اصلاً باز نمی‌شود
راه‌حل: مطمئن شوید مراحل "راه‌اندازی اولیه" را دنبال کرده‌اید

مشکل: خطای "App is damaged"
راه‌حل: 
  1. Terminal را باز کنید
  2. تایپ کنید: xattr -cr /Applications/ImageConverter.app
  3. Enter را بزنید
  4. دوباره برنامه را باز کنید

مشکل: تبدیل با شکست مواجه می‌شود
راه‌حل: 
  • مطمئن شوید تصاویر منبع معتبر و خراب نیستند
  • بررسی کنید که مجوز نوشتن در پوشه خروجی دارید
  • برنامه را ری‌استارت کنید
